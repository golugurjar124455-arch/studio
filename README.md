# CoinTrack Pro - डिप्लॉयमेंट और सेटअप गाइड

यह ऐप एक क्लाउड-पावर्ड निवेश पोर्टल है जिसे आप अपने मोबाइल पर मुफ़्त में चला सकते हैं।

## 💻 अपने कंप्यूटर पर फाइलों को कैसे रखें:
1. `cointrack-pro` नाम का एक फोल्डर बनाएं।
2. उसके अंदर `src`, `public`, `components` फोल्डर बनाएं।
3. AI द्वारा दी गई फाइलों को उनके सही पाथ (Path) पर सेव करें (उदा: `src/lib/db.ts`)।

## 🚀 GitHub पर अपलोड करें (मुफ़्त लिंक के लिए):
1. [GitHub](https://github.com/) पर जाएं और 'New Repository' बनाएं।
2. अपने फोल्डर की सभी फाइलों को वहां अपलोड करें।

## 🌐 HTTP लिंक प्राप्त करें (Vercel):
1. [Vercel](https://vercel.com/) पर जाएं।
2. अपनी GitHub रिपॉजिटरी को वहां 'Import' करें।
3. **Deploy** बटन दबाएं।
4. 1 मिनट में आपको `https://your-app.vercel.app` जैसा लिंक मिल जाएगा।

## 📱 मोबाइल पर ऐप कैसे इंस्टॉल करें:
1. अपने मोबाइल के Chrome ब्राउज़र में अपना Vercel लिंक खोलें।
2. ब्राउज़र की सेटिंग्स (तीन बिंदु) में जाएं।
3. **"Add to Home Screen"** चुनें।
4. अब यह आपके फोन में बिना ब्राउज़र के एक असली ऐप की तरह खुलेगा।
