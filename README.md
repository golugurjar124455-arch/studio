# CoinTrack Pro - प्रोफेशनल इन्वेस्टमेंट ट्रैकर

यह एक "Cloud-First" पीएफ़सी (PWA) ऐप है जिसे वित्तीय सलाहकारों के लिए बनाया गया है। अब यह Firebase Firestore का उपयोग करता है।

## इसे लाइव (Deploy) कैसे करें (How to Deploy for Free):
1. **GitHub पर कोड डालें:** अपने इस कोड को GitHub रिपोजिटरी में पुश करें।
2. **Firebase Console पर जाएं:** [console.firebase.google.com](https://console.firebase.google.com/) पर जाकर एक नया प्रोजेक्ट बनाएं।
3. **Build -> App Hosting:** Firebase में "App Hosting" मेनू पर जाएं।
4. **Connect GitHub:** अपना GitHub अकाउंट कनेक्ट करें और यह रिपोजिटरी चुनें।
5. **Environment Variables:** डिप्लॉयमेंट के समय अपनी Firebase Keys को `NEXT_PUBLIC_` प्रीफिक्स के साथ जोड़ें।
6. **Live Link:** डिप्लॉय होने के बाद Firebase आपको एक `web.app` वाला लिंक देगा जिसे आप कहीं भी शेयर कर सकते हैं।

## मोबाइल पर ऐप की तरह कैसे इस्तेमाल करें (Install as App):
1. अपने मोबाइल ब्राउज़र (Chrome/Safari) में डिप्लॉय किया गया लिंक खोलें।
2. ब्राउज़र की सेटिंग्स (तीन बिंदु ⋮ या Share बटन) पर क्लिक करें।
3. **"Add to Home Screen"** चुनें।
4. अब आपके मोबाइल की होम स्क्रीन पर CoinTrack Pro का आइकॉन आ जाएगा।

## मुख्य विशेषताएं:
- **Cloud Database:** आपका डेटा अब सुरक्षित क्लाउड में रहता है।
- **Bitcoin Style UI:** प्रोफेशनल डार्क मोड और ऑरेंज थीम।
- **Admin Panel:** जीएसटी, फीस और प्लेटफॉर्म सेटिंग्स पर पूरा कंट्रोल।
- **WhatsApp Reports:** क्लाइंट को उनके प्रदर्शन की रिपोर्ट भेजने की सुविधा।
