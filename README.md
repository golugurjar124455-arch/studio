# सुरक्षित निवेश ट्रैकिंग ऐप (डेमो)

यह एक "Local-First" निवेश ट्रैकिंग ऐप है जिसे वित्तीय सलाहकारों के लिए बनाया गया है।

## मुख्य विशेषताएं:
- **पूरी तरह से सुरक्षित:** आपका सारा डेटा आपके ब्राउज़र के `localStorage` में रहता है।
- **बिना इंटरनेट/API:** यह ऐप बिना किसी Gemini API key के काम करता है।
- **स्मार्ट विश्लेषण:** निवेश के प्रदर्शन का हिंदी में संक्षिप्त विवरण।
- **डैशबोर्ड:** क्लाइंट्स के प्रदर्शन का ग्राफिकल चार्ट।

## कैसे उपयोग करें:
1. ऐप के मुख्य पेज पर जाएं।
2. लॉगिन विवरण दर्ज करें:
   - **यूजरनेम:** `admin`
   - **पासवर्ड:** `admin`
3. डैशबोर्ड पर आप नए क्लाइंट जोड़ सकते हैं और पुराने क्लाइंट्स का विवरण देख सकते हैं।

## तकनीकी विवरण:
- **Framework:** Next.js 15
- **UI:** Tailwind CSS + ShadCN UI
- **Icons:** Lucide React
- **Charts:** Recharts
