# CoinTrack Pro - प्रोफेशनल इन्वेस्टमेंट ट्रैकर

यह एक "Local-First" पीएफ़सी (PWA) ऐप है जिसे वित्तीय सलाहकारों के लिए बनाया गया है।

## मोबाइल पर ऐप की तरह कैसे इस्तेमाल करें (Install as App):
1. इस ऐप को अपने मोबाइल के ब्राउज़र (Chrome/Safari) में खोलें।
2. ब्राउज़र की सेटिंग्स (तीन बिंदु ⋮ या Share बटन) पर क्लिक करें।
3. **"Add to Home Screen"** या **"Install App"** चुनें।
4. अब आपके मोबाइल की होम स्क्रीन पर CoinTrack Pro का आइकॉन आ जाएगा और यह बिल्कुल एक असली ऐप (APK) की तरह चलेगा।

## ऑनलाइन लिंक कैसे बनाएं (Deployment):
इस ऐप को पूरी दुनिया के लिए लाइव करने के लिए इन चरणों का पालन करें:
1. अपने कोड को GitHub पर पुश करें।
2. **Firebase Console** पर जाएं।
3. **App Hosting** सेवा चुनें और अपना GitHub रिपोजिटरी कनेक्ट करें।
4. डिप्लॉय होने के बाद आपको एक `web.app` वाला लिंक मिल जाएगा जिसे आप कहीं भी शेयर कर सकते हैं।

## मुख्य विशेषताएं:
- **Bitcoin Style UI:** प्रोफेशनल डार्क मोड और ऑरेंज थीम।
- **PWA Support:** बिना APK डाउनलोड किए मोबाइल ऐप की तरह इस्तेमाल करें।
- **Security:** सारा डेटा सुरक्षित स्थानीय स्टोरेज में।
- **Admin Panel:** जीएसटी, फीस और सिस्टम रिसेट पर पूरा कंट्रोल।
