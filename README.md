# CoinTrack Pro - प्रोफेशनल इन्वेस्टमेंट ट्रैकर

यह एक "Cloud-First" PWA ऐप है। इसे लाइव (HTTP लिंक पाने के लिए) करने के नीचे दिए गए तरीके अपनाएं।

## 🚀 मुफ़्त में लाइव (Deploy) कैसे करें:

### तरीका 1: Firebase App Hosting (सबसे अच्छा विकल्प)
1. **GitHub पर कोड डालें:** सबसे पहले अपने कोड को GitHub पर एक नई रिपॉजिटरी में पुश करें।
2. **Firebase Console खोलें:** [console.firebase.google.com](https://console.firebase.google.com/) पर जाएं।
3. **प्रोजेक्ट बनाएं:** "Add Project" पर क्लिक करें और अपने प्रोजेक्ट का नाम दें।
4. **App Hosting में जाएं:** बाएं हाथ के मेनू में 'Build' के अंदर 'App Hosting' चुनें।
5. **GitHub कनेक्ट करें:** "Get Started" दबाएं और अपने GitHub अकाउंट को कनेक्ट करके अपनी रिपॉजिटरी चुनें।
6. **Live Link:** सेटअप पूरा होने के बाद Firebase आपको एक लिंक देगा (जैसे `your-app.web.app`)। यही आपका HTTP लिंक है।

### तरीका 2: Vercel (सबसे आसान विकल्प)
1. [Vercel.com](https://vercel.com/) पर जाएं और GitHub से साइन अप करें।
2. 'Add New' -> 'Project' पर क्लिक करें।
3. अपनी GitHub रिपॉजिटरी को 'Import' करें।
4. 'Deploy' बटन दबाएं।
5. 1 मिनट के अंदर आपको एक `vercel.app` वाला लिंक मिल जाएगा।

## 📱 मोबाइल पर ऐप कैसे इंस्टॉल करें:
1. अपने मोबाइल के Chrome या Safari ब्राउज़र में अपना लाइव लिंक (जो डिप्लॉयमेंट के बाद मिला है) खोलें।
2. ब्राउज़र की सेटिंग्स (⋮ या Share बटन) पर जाएं।
3. **"Add to Home Screen"** विकल्प चुनें।
4. अब CoinTrack Pro आपके फोन में एक असली ऐप की तरह दिखेगा।

## 🔑 लॉगिन विवरण:
- **यूजरनेम:** `admin`
- **पासवर्ड:** `admin`

## ⚙️ एडमिन कंट्रोल:
लॉगिन करने के बाद नीचे दिए गए **Admin** (Shield Icon) बटन पर जाएं। वहां से आप:
- GST और UPI चार्ज बदल सकते हैं।
- प्लेटफॉर्म का नाम और करेंसी बदल सकते हैं।
- पूरा डेटा डिलीट (Wipe) कर सकते हैं।
