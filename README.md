# 📱 CoinTrack Pro - मोबाइल सेटअप और डिप्लॉयमेंट गाइड

यह ऐप विशेष रूप से मोबाइल पर निवेश ट्रैक करने के लिए बनाया गया है।

## 🚀 मुफ़्त HTTP लिंक (लिंक) कैसे प्राप्त करें:
चूंकि आप मोबाइल पर हैं, सबसे आसान तरीका **Vercel** का उपयोग करना है:
1. [GitHub.com](https://github.com/) पर जाएं (डेस्कटॉप साइट मोड में खोलें)।
2. 'New Repository' बनाएं और उसका नाम `investment-app` रखें।
3. AI द्वारा दी गई फाइलों को एक-एक करके वहां 'Create new file' बटन से पेस्ट करें।
4. [Vercel.com](https://vercel.com/) पर लॉगिन करें और अपनी GitHub रिपॉजिटरी को 'Import' करें।
5. **Deploy** बटन दबाएं। आपको `https://your-app.vercel.app` जैसा लिंक मिल जाएगा।

## 📲 मोबाइल पर ऐप कैसे इंस्टॉल करें (PWA):
1. अपने मोबाइल के **Chrome** ब्राउज़र में अपना Vercel लिंक खोलें।
2. ऊपर दाईं ओर **तीन बिंदुओं (⋮)** पर क्लिक करें।
3. **"Install App"** या **"Add to Home Screen"** चुनें।
4. अब यह आपके फोन के होम स्क्रीन पर एक असली ऐप (Icon) की तरह दिखेगा।
