# CoinTrack Pro - डिप्लॉयमेंट गाइड

## 🚀 GitHub पर अपलोड कैसे करें:
1. GitHub पर लॉग इन करें और 'cointrack-pro' नाम से **New Repository** बनाएं।
2. अपने कंप्यूटर पर इस प्रोजेक्ट की सभी फाइलें एक फोल्डर में रखें।
3. GitHub डेस्कटॉप या कमांड लाइन का उपयोग करके फाइलें अपलोड (Push) करें।

## 🌐 मुफ़्त में HTTP लिंक कैसे पाएं (Vercel):
1. [Vercel.com](https://vercel.com/) पर जाएं।
2. 'Add New' -> 'Project' चुनें।
3. अपनी GitHub रिपॉजिटरी को कनेक्ट करें।
4. **Deploy** बटन दबाएं।
5. 1 मिनट में आपको आपका **पब्लिक HTTP लिंक** मिल जाएगा।

## 📱 मोबाइल पर ऐप कैसे लगाएं:
1. अपने मोबाइल के ब्राउज़र में अपना Vercel लिंक खोलें।
2. ब्राउज़र मेनू (तीन बिंदु) में जाएं और **"Add to Home Screen"** चुनें।
3. अब यह आपके फोन में एक असली ऐप की तरह काम करेगा।
